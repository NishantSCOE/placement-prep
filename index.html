Sync setup — one time only

Connect a free Firebase database so your progress syncs between your phone and laptop automatically. Takes 5 minutes. Or skip to use locally on this device only.

  1. 1Go to console.firebase.google.com and sign in with your Google account
  2. 2Click Add project → name it placement-prep → disable Google Analytics → click Create project
  3. 3Inside the project, click the Web icon </> → register app as placement-prep → you will see a firebaseConfig block
  4. 4Go to Build → Firestore Database → Create database → Start in test mode → Enable
  5. 5Copy the firebaseConfig object from step 3 and paste it below
Example (yours will have different values): { apiKey: "AIzaSyXXXXXXXXXXXXXXXXX", authDomain: "placement-prep-12345.firebaseapp.com", projectId: "placement-prep-12345", storageBucket: "placement-prep-12345.appspot.com", messagingSenderId: "123456789012", appId: "1:123456789012:web:abcdef123456" }
